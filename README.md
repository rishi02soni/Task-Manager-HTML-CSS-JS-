# Task Manager Web App

A clean and responsive Task Manager built using **HTML, CSS, and JavaScript**. This project allows users to manage daily tasks with an intuitive interface and interactive controls.

---

## 🚀 Features

- Add New Tasks
- Mark Tasks as Completed
- Delete Tasks
- Responsive Modern UI
- Beginner Friendly Code Structure

---

## 📂 Project Structure

```bash
TaskManager/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🛠️ Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## ▶️ How to Run
- Clone the repository
```
git clone <your-repo-url>
```
- Open project folder
- Run index.html in browser
